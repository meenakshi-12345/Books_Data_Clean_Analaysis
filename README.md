⭐ Overview

This project explores how an author’s rating influences the commercial success of their books. Using a cleaned dataset containing book details, the analysis focuses on understanding sales trends across different author categories—Novice, Intermediate, Famous, and Excellent.

The goal is to uncover whether author reputation plays a significant role in driving book sales, and to visualize measurable differences between author groups using data-driven techniques.

🧹 Dataset & Preparation

The dataset was cleaned and processed to ensure accuracy. Key steps included:

Handling missing and inconsistent values

Converting data types

Removing duplicates

Standardizing categorical fields

Preparing numerical columns for visualizations

The cleaned dataset forms the foundation for all analysis and insights.

📊 Key Insights

Excellent-rated authors sell significantly more units compared to all other categories. Their sales distribution is broader, with many high-value outliers.

Novice, Intermediate, and Famous authors show lower, more stable sales ranges with limited variability.

The data suggests a strong link between author credibility and commercial success, indicating that higher author ratings attract greater readership and market demand.

📈 Visualizations

This project uses Matplotlib and Seaborn to generate visual insights, including:

Boxplots comparing units sold across author ratings

Category-wise sales distribution

Outlier inspection for high-selling titles

Comparative analysis of median and spread values

These visualizations make patterns easy to interpret and actionable.

🧠 Objective

To determine whether author ratings can be used as a predictive indicator for book performance and to help publishers, marketers, and analysts understand how author credibility shapes consumer behavior.
